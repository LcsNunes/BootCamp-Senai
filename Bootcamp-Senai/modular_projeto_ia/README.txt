# Projeto Modular - Classificação de Falhas em Chapas de Aço Inoxidável

## Estrutura de Pastas
- `dados/`: onde será salvo o arquivo `dados_tratados.csv` após o pré-processamento.
- `imagens/`: onde serão salvos todos os gráficos gerados durante o projeto.
- `main.py`: script principal que executa todas as etapas do pipeline.
- Demais arquivos `.py` contêm funções específicas de cada etapa do processo.

## Como Executar
1. Coloque o arquivo `bootcamp_train.csv` dentro da pasta `modular_projeto_ia/`.
2. Abra um terminal nessa pasta e execute:
   python main.py

## Requisitos
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
